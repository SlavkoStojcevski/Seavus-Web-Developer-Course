Some aditional info.
The "article" element encloses a block of related content that makes sense
on its own without the rest of the page (e.g., a single blog post)
The "div" element is used when the content has no semantic value, always starts on a new line and takes the whole width of a webpage.
The "span" element is an inline non-semantic element, starts on the same line and takes as much space as it needs.
